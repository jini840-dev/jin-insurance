[PRD] 청소년 타겟 마중물 서비스: 'JIN-Growth' 플랫폼
프로젝트 개요
10대 청소년들은 자산에 매우 관심이 많다.
내가 성인이 되었을때 얼마의 자산이 있는지, 어떻게 운영을 하면 좋은지, 또 독립이라는 한단계 성장으로 인해
주의해야 하는 자산관리 등에 관심이 많다.
이제 그 첫 걸음으로, 당사 앱을 통해 청소년이 자산을 지킨다는 것보다 내가 성장하면서 자산을 키우는 느낌을 주고 싶다.
자산키우기 방법으로 여러가지 미션실행을 통해 포인트로 치환될 예정이다.

1.첫페이지


- 10대 청소년들이 접속했을때 친숙한 힙합 페이지 구성.
- 복잡한 로그인 과정 보다 카카오계정으로 자동 로그인 구성.
- 배경음악도 힙하게 구성.

2.로그인
- 카카오계정 또는 휴대폰 번호 인증으로 구성.
- 수집정보는 카카오계정 또는 휴대폰번호와 닉네임.

3.로그인 이후 페이지 구성
- 사용자가 현재 진행중인 미션 리스트 구성
  .. 미션카드로 보여주기
  .. 미션의 시작일자~종료일자
  .. 미션의 성공율(상태바 제공)
  .. 획득한 포인트와 앞으로 미션 수행하면 획득하게 되는 포인트(동기부여)
  .. 성인이 되었을때, 이미 획득한 포인트로 복리이자 계산된 자산 보여주기

4. 미션종류 등록하게 페이지 구성(운영자권한)
- 공부/독서 타이머 챌린지
- 행운자산화 퀴즈풀기 챌린지
- 효도활동 자산화 챌린지

우선 여기까지 화면 페이지 구성을 해죠.
페이지는 너가 필요한 만큼 만들면 좋겠어.
